## Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/lokeshdarla/TechSynergy
    ```

2. **Navigate to the project directory:**

    ```bash
    cd TechSynergy
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

## Usage

```bash
  # to run the project
```
